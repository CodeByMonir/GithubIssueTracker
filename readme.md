1️⃣ What is the difference between var, let, and const?

Ans : => var, let and const in JavaScript differ mainly in scope and reassignability.
*var is function-scoped or global, hoisted, and re-declarable.
*let and const are block-scoped(scoped to {}), with let allowing reassignment.
*const prohibiting it. const requires initialization, while let and var do not.

2️⃣ What is the spread operator (...)?

Ans : => JavaScript is an ES6 feature that expands iterables lie arrays, strings or object properties into individual elements. It is primarily used for shallow copying, merging arrays or objects, and passing arguments to functions, allowing for cleaner code and maintaining immutability.


3️⃣ What is the difference between map(), filter(), and forEach()?

Ans : => The primary difference between map(), filter() and forEach() in JavaScript is their purpose and return value.
*map() transforms each element and returns a new array of the same length.
*filter() selects a subset of elements that meet a condition and returns a new array.
*forEachJ() simply executes a function for each elements with no return value.


4️⃣ What is an arrow function?

Ans : => Arrow functions, introduced in ES6, provide a concise syntax for writing JavaScript functions using the => (arrow) operator. They are anonymous, lack their own this or arguments, and cannot be used as constructors. Ideal for callbacks, they implicitly return values if used without curly braces.


5️⃣ What are template literals?

Ans : => Template literals are a JavaScript ES6 feature that allows for easier string interpolation, multi-line strings, and embedded expressions using backticks (`) instead of quotes. They simplify code by allowing variables or expressions to be placed directly within a string using the ${expression} syntax, avoiding complex concatenation.
